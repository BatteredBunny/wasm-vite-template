<h1 align="center">wasm-vite-template</h1>

<p align="center">An opinionated template for using rust wasm with vite</p>

```
cargo generate --git https://github.com/ayes-web/wasm-vite-template
```

## Nix

```
git add *.nix
```

### Build
```
nix build
```

### Shell
```
nix develop
```

## Makefile usage

### Build

```
make
```

### Start dev server

```
make dev
```

### Preview production build

```
make start
```
